## ملخص التقنيات المعتمدة

| الطبقة | التقنية | السبب الرئيسي |
|--------|---------|---------------|
| الواجهة الأمامية | React 19 + Vite + shadcn/ui | الأكثر انتشاراً، HMR سريع |
| الترجمة | react-i18next + RTL/LTR | عربي + إنجليزي بسلاسة |
| طبقة API | Express 5 + Node.js | TypeScript مشترك، ناضج |
| عقد API | OpenAPI 3.1 + Orval | تطابق تام واجهة/خادم |
| قاعدة البيانات | PostgreSQL + Drizzle ORM | type-safe، أداء عالي |
| الذكاء الاصطناعي | OpenAI / Anthropic (قابل للتبديل) | نماذج متعددة القوة |
| المصادقة | Replit Auth (OIDC + PKCE) | جاهز وآمن |
| قائمة المهام (Queue) | Redis + BullMQ | in-memory سريع، retry، مراقبة |
| الدفع | Stripe | الأكثر موثوقية عالمياً |
| التحقق | Zod v4 | type-safe validation |
